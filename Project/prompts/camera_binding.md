### 需求
编写树莓派相机标定图片保存程序。

### 交付要求
- 一个 Python 程序 A，实现一个网页，在网页上可控制树莓派所连相机拍照。
- 另一个 Python 程序 B，可将树莓派拍得的照片通过 Tailscale 传输至电脑端保存。

### 部署位置
代码部署在树莓派上。

### 参考程序
- integrated_pipeline.py，其中包含了借助 Tailscale 将树莓派摄像头画面截取到本机保存的程序。
- ws_car_avoid.py 是树莓派端的运行程序，通过网页对外开放视频流和运动控制 API，可在网页上点击截取视频页面。

### 程序实现详细要求
1. 对于 Python 程序 A，创建一个类似于 ws_car_avoid.py 中创建网页的网页，只保留显示摄像头画面和点击截取视频页面的功能。
2. 对于 Python 程序 B，可以将截取的视频页面传输到本主机，并保存到参数指定的本主机路径下。
