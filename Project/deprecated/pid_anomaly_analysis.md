附件：newrun.ino 以及 3 份异常串口日志

需要开发基于 Mega2560 板、MPU-6500 的转向 PID 控制程序。附件中的 newrun.ino 在执行时 Yaw 参数和 GyroZ 参数异常，怀疑为陀螺仪硬件问题。

工作要求：
1. 查询网络，寻找采用相同硬件配置，实现小车转向、直行运动的项目案例。可以寻找 GitHub 或其他硬件开发社区。
2. 搜索网络，寻找是否有相似的参数异常案例，分析导致上述异常参数输出的可能原因。
3. 综合上述分析，指出在开发所需转向控制 PID 算法过程中的缺陷和不足，给出可能的解决方案（完全重写一份新的 .ino 文件，或者在 newrun.ino 基础上进行一定修改使其可用）。

已知的硬件特性
- 电机在输出功率为 150 左右时才可开始运动，若低于该功率，可能无法使轮子转动

现有的供电策略
- Arduino 板通过树莓派连出的 USB 线供电
- 四轮电机主要通过外接电池盒供电
